Eternium API Reference

A unified REST API for AI image generation, video generation, chat, embeddings, and audio. Built on Cloudflare Workers. Global edge. 200 credits per dollar.

Base URL https://api.eternium.ai
Authentication

Two auth mechanisms depending on the endpoint type.

🔑
API Key: Pass as X-API-Key: etrn_... or Authorization: Bearer etrn_.... Used for all generation, usage, media, and content endpoints.
⚠
Supabase JWT: Authorization: Bearer <jwt>. Used only for account management (checkout, provision-key). Issued by Supabase Auth on signup.
shell
# API Key header
curl https://api.eternium.ai/v1/usage \
  -H "X-API-Key: etrn_your_key_here"

# Bearer token (equivalent)
curl https://api.eternium.ai/v1/usage \
  -H "Authorization: Bearer etrn_your_key_here"
Errors

All errors return JSON with a consistent shape.

json
{
  "error":  "Insufficient credits",
  "code":   "INSUFFICIENT_CREDITS",
  "status": 402
}
StatusCodeMeaning
400BAD_REQUESTMissing or invalid parameters
401UNAUTHORIZEDNo API key provided
402INSUFFICIENT_CREDITSAccount balance too low
403FORBIDDENKey exists but lacks permission (admin routes)
404NOT_FOUNDResource not found
429RATE_LIMITEDToo many requests; retry after the Retry-After header
500UPSTREAM_ERRORUpstream provider (Kie.ai / OpenAI) failed
Credits

All paid operations consume credits. 1 credit = $0.005 (200 credits per dollar). Credits never expire.

💰
Image generation: ~10-50 credits per request depending on model. Video generation: ~100-500 credits. Chat: ~1-5 credits per 1k tokens. Embeddings: ~1 credit per 1k tokens.
Public Endpoints

No authentication required.

GET/health publicService status

Returns current service health, upstream provider status, and build metadata.

response
{ "status": "ok", "version": "1.0.0", "upstream": "kie.ai" }
GET/v1/models publicAll 21 models

Returns the full model catalog. 7 image, 8 video, 3 chat, 2 embedding, 1 audio models.

response
{ "models": [{ "id": "nano-banana-2", "type": "image", "credits_per_request": 12 }] }
GET/v1/models/featured publicFeatured models

Returns only featured models recommended for new users.

GET/v1/tiers publicPricing tiers

Returns available subscription tiers with credit allowances, rate limits, and feature flags.

GET/v1/pipelines publicAvailable pipelines

Returns available multi-step pipelines: product-shot, social-media-pack, video-ad, thumbnail-pack.

GET/v1/products publicProduct catalog

Returns the Armory product catalog (AI Builder's Playbook series). Frontends use this to stay in sync without code deploys.

response
{ "products": [{ "slug": "claude-kit", "episode": "EP1", "requiresAuth": true }] }
GET/v1/content/blog publicPublished blog posts

Returns all published blog posts from the Supabase content backend.

GET/v1/content/blog/:slug publicSingle post

Returns a single blog post by slug including full content, metadata, and related posts.

Account

Supabase JWT required. These endpoints handle payment and key provisioning after Supabase signup.

POST/auth/checkout jwtStripe checkout

Creates a Stripe checkout session for a subscription tier. Returns a URL to redirect the user.

ParameterTypeRequiredDescription
tierstringrequiredstarter or pro
success_urlstringoptionalPost-payment redirect URL
POST/auth/provision-key jwtGet API key

Issues an API key after payment confirmation. Returned exactly once -- store it securely.

response
{ "key": "etrn_...", "tier": "starter", "credits": 5000 }
POST/auth/regenerate-key jwtRotate API key

Invalidates the current key and issues a new one. Old key stops working immediately. All credits transfer.

Generation

Async image and video generation. Submit, get a task ID, poll for results.

POST/v1/generate api keyGenerate image or video

Submit a generation request. Returns a task ID immediately. Poll /v1/tasks/:id until status: "completed".

ParameterTypeRequiredDescription
modelstringrequiredModel ID (e.g. nano-banana-2, veo-3, sora-2)
promptstringrequiredText prompt describing the desired output
negative_promptstringoptionalWhat to exclude from the generation
widthnumberoptionalOutput width in pixels (default: 1024)
heightnumberoptionalOutput height in pixels (default: 1024)
durationnumberoptionalVideo duration in seconds (video models only)
request
curl -X POST https://api.eternium.ai/v1/generate \
  -H "X-API-Key: etrn_..." \
  -H "Content-Type: application/json" \
  -d '{
    "model": "nano-banana-2",
    "prompt": "Futuristic city at night, neon lights",
    "width": 1024, "height": 1024
  }'
response
{ "task_id": "tsk_abc123", "status": "queued", "credits_used": 12 }
GET/v1/tasks/:id api keyPoll task status

Poll for generation results. Status values: queued, processing, completed, failed. When completed, includes output URL and permanent R2 URL.

response
{
  "task_id":   "tsk_abc123",
  "status":    "completed",
  "output_url": "https://cdn.kie.ai/...",
  "r2_url":    "https://api.eternium.ai/media/generations/tsk_abc123"
}
GET/v1/tasks/:id/download api keyDownload URL

Returns a direct download URL for the completed task. Automatically archives the result to R2 for permanent storage.

GET/v1/usage api keyCredit balance

Returns credit balance, usage totals, and spending breakdown for the authenticated key.

response
{
  "credits_remaining": 4120,
  "credits_used":      880,
  "total_generations": 73,
  "tier":               "starter"
}
Chat

OpenAI-compatible chat completions. Drop-in replacement -- change the base URL.

OpenAI-compatible chat completions with optional streaming. Models: gpt-5.1, gpt-5.1-codex-mini, gpt-5.4.

request
{
  "model":    "gpt-5.1",
  "messages": [{"role": "user", "content": "Hello"}],
  "stream":   false
}
Embeddings

OpenAI-compatible text embeddings for semantic search and RAG pipelines.

POST/v1/embeddings api keyText embeddings

Generate vector embeddings. Models: text-embedding-3-small (1536d), text-embedding-3-large (3072d).

request
{ "model": "text-embedding-3-small", "input": "The quick brown fox" }
Audio

Whisper transcription via OpenAI-compatible endpoint.

POST/v1/audio/transcriptions api keyWhisper transcription

Transcribe audio using Whisper (whisper-1). Accepts multipart/form-data with an audio field. Returns plain text transcript.

request
curl -X POST https://api.eternium.ai/v1/audio/transcriptions \
  -H "X-API-Key: etrn_..." \
  -F "audio=@recording.mp3" \
  -F "model=whisper-1"
Media

Upload, serve, and delete files in Cloudflare R2. Generation outputs auto-archive here.

PUT/v1/media/upload api keyUpload to R2

Upload a file to Eternium's R2 storage. Returns a permanent URL. Supports images, video, audio, PDF, and other binary formats.

GET/v1/media/:key api keyServe from R2

Retrieve a file from R2 by key. Generation outputs at /media/generations/:task_id are served without auth.

DELETE/v1/media/:key api keyDelete from R2

Delete a file from R2. Only files uploaded by your API key can be deleted. Returns 204 on success.

Pipelines

Multi-step automated workflows -- complex sequences in a single call.

POST/v1/pipelines/run api keyRun a pipeline

Run a named pipeline. Available: product-shot (3 product angles), social-media-pack (3 aspect ratios), video-ad (hero + animated video), thumbnail-pack (4 variations).

request
{
  "pipeline": "product-shot",
  "prompt":   "Black leather luxury watch on obsidian surface"
}
POST/v1/thumbnails/generate api key3 thumbnail concepts

Generate 3 campaign-aware YouTube thumbnail concepts (A: face+logos+text, B: stat/number hero, C: visual metaphor). Returns task IDs for polling.

request
{
  "title":         "I Built an AI SaaS in 48 Hours",
  "hook":          "Zero to $1k MRR",
  "key_takeaways": ["No code", "Claude Code agent"],
  "content_pillar": "business",
  "model":          "nano-banana-2"
}
Admin

Requires an API key whose email matches ADMIN_EMAIL on the worker.

GET/admin/overview adminDashboard metrics

Returns aggregated metrics: users, MRR, revenue, credits, generations, upstream costs, model breakdown, user list, health, and alerts.

GET/admin/tenants adminList tenants

Returns all managed hosting tenants with subdomain, status, and usage data.

POST/admin/users/:email/revoke adminSuspend user

Suspends a user's API key immediately. Reversible via /activate.

POST/admin/users/:email/activate adminReactivate user

Reactivates a suspended user's API key.

POST/v1/content/blog/publish adminPublish blog post

Publishes or updates a blog post in the Supabase content backend. Admin only.

JavaScript SDK

Drop-in client for Node.js and browser. Located at sdk/js/ in the repo.

install
npm install eternium-sdk
usage
import { EterniumClient } from 'eternium-sdk';

const client = new EterniumClient({ apiKey: 'etrn_...' });

const task = await client.generate({
  model:  'nano-banana-2',
  prompt: 'Futuristic cityscape at dusk'
});
const result = await client.poll(task.task_id);
console.log(result.output_url);
Python SDK

Sync and async clients for Python 3.8+. Located at sdk/python/ in the repo.

install
pip install eternium-sdk
usage
from eternium import EterniumClient

client = EterniumClient(api_key="etrn_...")
task   = client.generate(model="nano-banana-2", prompt="Futuristic cityscape")
result = client.poll(task["task_id"])
print(result["output_url"])
